Project Overview

This repository hosts the isolated GlobeWest 2026 front-end code, test scripts, and documentation. It separates the GlobeWest storefront from the main FLM project to avoid directory conflicts.

🔑 Core Pillars of Accessibility

  • Color Contrast (Criteria 1.4.3): Clear 4.5:1 ratio for all body copy.
  • ARIA & Text Labels (Criteria 1.1.1): Clear accessible labels on all controls.
  • Touch Target Sizing (Criteria 2.5.8): Minimum 44x44 CSS pixels for interactive elements.
  • Scalability (Criteria 1.4.4): Free browser-level zooming enabled.

Our automation suites are run using Playwright and Axe-Core to verify compliance programmatically on the mcstaging2.globewest.com.au staging server.
